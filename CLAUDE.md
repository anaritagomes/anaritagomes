## Repository Purpose

This is a [GitHub special profile repository](https://docs.github.com/en/account-and-profile/setting-up-and-managing-your-github-profile/customizing-your-profile/managing-your-profile-readme). The `README.md` in this repository automatically appears on the GitHub profile page for the user `anaritagomes`.

## Structure

- `README.md` — The profile README rendered on the GitHub profile page. This is the only meaningful file and the primary thing to edit.

## Working on This Repo

There are no build steps, dependencies, or tests. Changes to `README.md` take effect on the GitHub profile once pushed to the `main` branch.

GitHub profile READMEs support standard Markdown and also render certain GitHub-flavored features such as:
- Badge widgets (e.g. from shields.io)
- GitHub stats cards (e.g. from github-readme-stats)
- Embedded images and GIFs
- HTML for layout control (e.g. `<div align="center">`)
